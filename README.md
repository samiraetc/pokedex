## Quick Start

Clone the repository to your local computer

```
https://github.com/samiraetc/pokedex.git
```

Follow `package.json` commands to assist in development:

- To install dependencies 

```
npm install
```

- To run application locally

```
npm run dev
```

```
http://localhost:5173/
```
